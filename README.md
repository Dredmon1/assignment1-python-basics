# Assignment 1 – Python Basics

## How to run
Option A (Docker – recommended)
```bash
docker build -t assignment1 .
docker run --rm assignment1
```

Option B (Local Python – optional)

```bash
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
